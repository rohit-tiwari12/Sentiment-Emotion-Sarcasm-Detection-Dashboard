# Sentiment + Emotion + Sarcasm Detection Dashboard

This project contains:

✔ FastAPI backend  
✔ Streamlit dashboard  
✔ Sentiment analysis  
✔ Emotion detection  
✔ Sarcasm detection  
✔ Topic modeling  
✔ Preprocessing pipeline  

## Run Backend
uvicorn api.app:app --reload

## Run Dashboard
streamlit run dashboard/app.py
